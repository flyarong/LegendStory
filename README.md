## 使用到的工具

Cocos2d-x 3.16 一款利用C++開發的遊戲引擎(開放原始碼)

Cocos2dStudio v3.0 可以設計遊戲介面

Physics-body-editor 用來設置地圖物理引擎的工具

DragonBones龍骨 用來做骨骼動畫的工具

DragonBonesCpp 龍骨官方提供對骨骼檔案解析的C++ Library(開放原始碼)
